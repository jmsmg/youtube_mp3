# youtube_mp3
> 유튜브 음원 추출 프로그램

## 가이드

1. 소스코드를 내려 받는다.
2. pip install pytube를 터미널 창에 입력하여 설치한다.
3. 터미널 창에 python main.py로 실행
4. URL 입력 및 엔터
5. 종료시 컨트롤 + C
